# Mentor Session Template

Use this for every live mentorship session.

## Session Details

- Date:
- Day number:
- Topic:
- Project or file worked on:

## 1. Warm-Up

Ask:

- What did you learn yesterday?
- What felt confusing?
- What did you try on your own?

## 2. Simple Analogy

Explain today's topic using a non-technical comparison.

Example:

> Git is like a save system for code. GitHub is like shared cloud storage for those saves.

## 3. Mentor Demo

Build a tiny example while speaking out loud.

Checklist:

- Explain the problem first.
- Write the smallest possible working version.
- Show one mistake and how to debug it.
- Ask the mentee to predict what happens next.

## 4. Guided Practice

Let the mentee drive.

Your role:

- Ask questions.
- Give hints before answers.
- Let them read error messages.
- Let them search documentation when appropriate.

## 5. Independent Practice

Give them a small task that is similar but not identical to the demo.

They should complete it without copying the exact code.

## 6. Homework

Assign:

- One video section or reading
- One written explanation
- One coding task
- One research question

## 7. Exit Check

Ask three questions:

1. What problem does this tool or concept solve?
2. Can you explain it using an analogy?
3. Can you build a tiny example without looking at the demo?
