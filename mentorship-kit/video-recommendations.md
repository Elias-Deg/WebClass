# YouTube And Research Recommendations

Use these videos as support, not as the whole lesson. The mentee should watch actively: pause, code along, write notes, and explain the idea back in simple words.

Links were selected for beginner usefulness and checked while preparing this kit on 2026-05-11. For framework APIs, always compare the video with the official documentation because tools change.

## How To Assign Videos

For each day:

1. Give only one main video or section.
2. Ask for a 5-bullet summary.
3. Ask them to copy one example by hand.
4. Ask them to build a tiny variation without copying.
5. Review their notes before moving on.
