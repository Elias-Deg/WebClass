# Mentor Portal

Open `index.html` in a browser to use the mentorship dashboard.

This is a plain HTML, CSS, and JavaScript site. It does not need a server, build tool, package manager, or installation step.

## What It Includes

- 12-week mentorship overview
-example
- 60 day-by-day lesson plans
- YouTube and research links
- Copyable code samples inside each relevant lesson
- Dedicated code sample library
- Homework for every day
- Tests for each stage
- Final readiness rubric
- Browser-saved progress and mentor notes

## Notes

Progress and mentor notes are stored in the browser with `localStorage`. If you open the site in a different browser or clear site data, those notes will not follow automatically.
